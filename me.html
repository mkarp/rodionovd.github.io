Projects

  • rd_route
    A library for function hooking on OS X — like mach_override, but without assembly trampolines and hardcoded stuff. It’s built on top of Mach and dyld APIs.

  • liblorgnette
    Remote symbolication on both OS X and iOS.
    You may think of it as of dlsym(), but for any process running on your system.

  • task_vaccine
    Lightweight code injection library.
    task_for_pid(), handling exceptions in a remote process, modifying a thread state, and all that.

  • SWRoute
    First PoC of function hooking in Swift.

  • machobot
    A handy Python toolkit for Mach-O files static analysis.

  • Locatr
    Cocoa application for faking your geolocation on Mac. Wonna tweet from the North Pole? Give it a chance.

  • ABetterPlaceForTweetbot
    A positivity filter for Tweetbot in the morning. Inspired by @orta and abetterplace.
    Using DYLD_INSERT_LIBRARIES and Objective-C method swizzling for filtering tweets in your timeline.

  • Cegta
    Everything-in-a-header DSL for TDD/BDD written in C. Inspired by Specta and RSpec.
